ShotSpot is the first Instagram-based application that provides users with the best photography hotpots. The web app is build on Node.js and interfaces with the Google Maps API to deliver real-time top IG posts in the searched city.

💻: BeautifulSoup4, Python, Express.js, HTML5/CSS3/JS, JSON, Node.js, Google Maps API

🏆: Best Use of AWS, Most Promising, Best Domain

CalorEats is an Android calorie tracker that uses Computer Vision. Simply capture an image of your food item and the app will use the Nutritionix API to track the calorie and nutritional value of the product. Forgot to take an image of your food item before you ate it? No worries, users can also search their food item.

💻: Google Vision API, Framework7 (HTML5/CSS3/JS), firebase, jQuery, Node.js, Nutrionix API
